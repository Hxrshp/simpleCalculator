# Simple Calculator Web App

A clean and responsive calculator built using **HTML, CSS, and JavaScript**.
This project performs basic arithmetic operations with improved user experience and UI design.

---

## 🚀 Features

* ➕ Addition, ➖ Subtraction, ✖️ Multiplication, ➗ Division
* 🧠 Smart operator handling (e.g., `5+-` → `5-`)
* 🧹 Clear and delete functionality
* 🎨 Modern glass UI design
* ⚡ Fast and lightweight
* 📱 Responsive layout

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## 📂 Project Structure

```
simpleCalculator/
│── index.html
│── style.css
│── script.js
```

## ▶️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/Hxrshp/simpleCalculator.git
```

2. Open the folder

3. Run:

* Open `index.html` in your browser
  OR
* Use Live Server in VS Code
